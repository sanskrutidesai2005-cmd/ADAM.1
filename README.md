🩺 ADAM – AI-Based Ailment Detection & Medicine Suggestion App

📌 Project Overview

ADAM is a healthcare-based web application designed to assist users in understanding their symptoms and receiving general health guidance. The system analyzes user-input symptoms and provides possible conditions, basic recommendations, and next steps, along with additional features like AI chat support and medicine reminders.

⚠️ Disclaimer: This application provides general health information only and is not a substitute for professional medical advice.

⸻

🚀 Features

🔍 Symptom Checker

* Users can enter or select symptoms
* System provides:
    * Possible health conditions
    * Severity level
    * Confidence level
    * Description and precautions

⸻

🤖 AI Health Chatbot

* Interactive chatbot for basic health queries
* Provides:
    * Possible causes
    * Recommendations
    * Follow-up questions
    * Next steps guidance

⸻

💊 Medicine Suggestion & Reminder

* Suggests commonly used medicines based on symptoms
* Allows users to set medicine reminders

⸻

🌐 Multi-Language Support

* Available in:
    * English
    * Hindi
    * Marathi
* Improves accessibility for diverse users

⸻

🏥 Emergency Contacts

* Quick access to emergency services
* Clickable contact options

⸻

👤 User Profile & History

* Stores user details (age, gender)
* Tracks previously checked symptoms

⸻

🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* Framework: React (Vite)
* Styling: Tailwind CSS
* Backend (Basic Logic): JavaScript-based logic
* Database: JSON / Local data
* Deployment: Netlify
* Version Control: GitHub

⸻

⚙️ How It Works

1. User logs into the system
2. Inputs symptoms or uses AI chatbot
3. System analyzes input using symptom-disease mapping
4. Displays:
    * Possible diagnosis
    * Severity & confidence
    * Recommendations
5. User can:
    * Set medicine reminders
    * Check emergency contacts
